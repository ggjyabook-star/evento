# evento

Propuesta comercial interactiva y landing de cotización para los servicios de streaming y cobertura audiovisual de **MAW Soluciones**.

## Contenido del repositorio
- `index.html`: Estructura y contenidos de la propuesta comercial.
- `style.css`: Estilos visuales con diseño responsivo, tema oscuro y optimización para impresión/PDF.
- `app.js`: Lógica interactiva para la cotización y exportación de archivos.
- `phone_hero.png`: Imagen ilustrativa del smartphone de última generación en el evento.
